更换主题

  • 当前主题：Chic
  • 访问 https://hexo.io/themes 并下载主题至 themes 文件夹下
  • 修改博客根目录下的 _config.yml 文件，替换原 theme 内容
  • 运行 yarn clean 后预览

部署

推送代码至 origin/master 分支即可触发 cloud build 自动构建并部署在 cloud run 环境中。
